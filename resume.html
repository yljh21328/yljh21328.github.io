自傳

我是楊凱甯，來自彰化，目前就讀於國立中山大學資訊工程學系碩士班。主要研究目標為空間關鍵字搜尋，研究方向為改進現有架構及演算法，並嘗試找出更多資訊。預計畢業的時間為104年6月。

從大學三年級開始，因為做專題的緣故進了知識與資訊發掘實驗室。與老師學習做研究的方式並開始接觸資料探勘以及資訊檢索這方面的研究。進而開始接觸 PHP、mySQL 與 javascript 這些網頁開發相關的技術。而機運下擔任老師的研究助理，著手幫老師開發實驗用的網站。

而大四時因課業告一個段落，想嘗試到公司實習累積經驗，因而參加了 KKBOX 的實習面試，在與吳經理面試之後，順利的成為網站開發部的實習生。而在實習期間前半段主要負責 KKBOX intranet 的開發與維護，後半段則主要開發一些操作資料的工具程式。在這段期間學習到許多開發技巧以及工作所必須遵守的標準。像是學習使用 git 以及 vim 其他開發者工具就對於我往後開發程式的影響很大。

在研究所時期因為之前所接觸的領域大多是資料探勘以及資訊檢索方面的研究，於是我進入了資料庫系統實驗室。在這裡學長姐、同學以及老師都給我相當大的鼓勵與幫助，我很快的就找到了自己有興趣的東西，也就是空間關鍵字搜尋，於是開始著手一系列的研究，並開始發想更好的架構以及演算法來改進現有的方法。在經過實習的洗禮之後，我更熱衷於學習技術以及參加社群活動。我除了努力完成自己的研究之外，在課餘時間我會尋找高雄地區的開發者聚會活動來參加。最常參加的就是 KSDG 的聚會，而大型的活動 COSCUP 也有參加。我認為參加社群的好處是可以認識許多厲害的人，同時也可以學習新的技術以及許多技巧。

一路上走來，我認識了許多厲害的工程師，也看見大家對於的熱情。我希望憑藉著我本身的些許經驗，以及對於學習的熱誠，有一天也能擠身變為厲害的人。
